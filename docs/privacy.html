Support

Need help with Fall, Number… Fall!? You are in the right place.

How to Play

  • Choose Beginner or Expert mode from the main menu.
  • Move the falling number tile left or right with horizontal swipes.
  • Swipe down or double-tap to drop the tile faster.
  • Create horizontal or vertical sums that match the current Target number.
  • When the numbers add up correctly, those tiles clear from the board.
  • The game ends when new tiles can no longer enter the board.

Game Modes

Beginner mode is designed for easier learning and clearer visual guidance.
Expert mode starts faster, with randomized targets and numbers for a harder challenge.

Power-Ups

  • Row Clear removes a full row.
  • Column Clear removes a full column.
  • Reorder rearranges the numbers currently on the board.

Troubleshooting

  • If the game feels paused, check whether the pause menu is open.
  • If sound or haptics are not working, check the Settings screen inside the app.
  • If something does not look right, close and reopen the app.
